```powershell

```


如果只检查提交文件规则本身的内置样例：

```powershell
.\scripts\verify-release-files.ps1 -SelfTest
```

```powershell
.\scripts\final-audit.ps1
```

## 报告写作口径

- 报告中的实验数字只能来自 `results/main_benchmark_with_consistency_precheck_20260618_173045/benchmark.json` 或之后重新归档且通过 `verify-results.ps1` 的结果。
- 不要从 README、前端截图或口头记忆复制实验数字。
- 必须同时写 `case_count` 和 `unique_case_count`。
- `confirm` 应表述为可恢复人工确认，不应表述为静默放行。
- 一致性 precheck 是可选门控，不是默认 API 行为。
- execution proxy 是受控工具代理，不是 syscall 级 sandbox。
