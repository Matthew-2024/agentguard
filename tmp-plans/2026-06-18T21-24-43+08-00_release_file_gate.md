## Gate Coverage

`verify-release-files.ps1` reads Git-visible files using:

```powershell
git -c core.excludesFile= -c core.quotepath=false ls-files --cached --others --exclude-standard
```

It checks:

- Required backend, benchmark, frontend, Docker, script, result, and documentation files are visible to Git.
- Forbidden runtime or secret carrier files are not visible to Git.
- At least one reportable benchmark artifact exists under `results/*/benchmark.json`.
- At least one stage summary exists under `tmp-plans/*.md`.
